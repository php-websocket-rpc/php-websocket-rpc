# php-websocket-rpc

Async RPC over WebSocket for PHP and TypeScript.

## Packages

| Package | Type | Description |
|---------|------|-------------|
| [php-websocket-rpc/rpc](src/Rpc) | PHP | Core data model, serialization, middleware pipeline, and contract attributes |
| [php-websocket-rpc/rpc-client](src/RpcClient) | PHP | Async WebSocket RPC client with contract proxies |
| [php-websocket-rpc/rpc-server](src/RpcServer) | PHP | Async WebSocket RPC server with rate limiting |
| [@php-websocket-rpc/client](src/TypescriptRpcClient) | npm | TypeScript/JavaScript RPC client (browser + Node.js) |
| [@php-websocket-rpc/codegen](src/TypescriptCodegen) | npm | TypeScript interface generator from PHP contracts |
